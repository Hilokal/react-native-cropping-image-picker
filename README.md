# react-native-cropping-image-picker

Cropping and Picking Images

## Installation

```sh
npm install react-native-cropping-image-picker
```

## Usage

```js
import { multiply } from 'react-native-cropping-image-picker';

// ...

const result = await multiply(3, 7);
```
